### Header

<ParamField header="x-api-key" type="string" required={true}>
    Parameter defines the Retailed private key to use.
</ParamField>

### Parameters

<ParamField query="query" type="string" optional={false} placeholder="nike-air-max-1-travis-scott-wheat">
    Product query : `air-jordan-3-retro-white-cement-reimagined` or `c318bbcc-312a-4396-9252-698c203d1dea`
</ParamField>

### Response

<ResponseField name="object" type="Product Object">
    <Expandable title="properties" defaultOpen={true}>
        <ResponseField name="results" type="list">
            360 product images
        </ResponseField>
    </Expandable>
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://app.retailed.io/api/v1/stockx/product/360?query=nike-air-max-1-travis-scott-wheat' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <token>' \
```
