---
title: "Statistics"
api: "GET https://app.retailed.io/api/v1/sneakers/prices_stats"
description: "Get statistics for sneaker prices"
---

### Header

<ParamField header="x-api-key" type="string" required={true}>
    Parameter defines the Retailed private key to use.
</ParamField>

### Query Parameters

<ParamField query="sku" type="string" optional={true} placeholder="DV6773-220">
    Sneaker code SKU
</ParamField>

<ParamField query="size_eu" type="string" optional={true} placeholder="39.5">
    Sneaker size in EU
</ParamField>

<ParamField query="size_us" type="string" optional={true} placeholder="6">
    Sneaker size in US
</ParamField>

<ParamField query="size_jp" type="string" optional={true} placeholder="24.5">
    Sneaker size in JP
</ParamField>

<ParamField query="size_uk" type="string" optional={true} placeholder="6">
    Sneaker size in UK
</ParamField>

### Response

<ResponseField name="data" type="object">
    Statistics for sneaker prices
</ResponseField>

<ResponseField name="error" type="object">
    Not empty if an error occured
</ResponseField>

<ResponseField name="usage" type="object">
    API usage
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://app.retailed.io/api/v1/sneakers/prices_stats?sku=DH6927-140&size_us=6' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <token>' \
```

</RequestExample>
