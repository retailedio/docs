### Header

<ParamField header="x-api-key" type="string" required={true}>
    Parameter defines the Retailed private key to use.
</ParamField>

### Response

<ResponseField name="count" type="number">
    Indicates the number of platforms returned
</ResponseField>

<ResponseField name="data" type="object">
    Platforms data
</ResponseField>

<ResponseField name="error" type="object">
    Not empty if an error occured
</ResponseField>

```bash Example Request
curl --location --request GET 'https://app.retailed.io/api/v1/sneakers/platforms' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <token>' \
```
