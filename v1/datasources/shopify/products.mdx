title: "Products"
api: "GET https://app.retailed.io/api/v1/shopify/products"
description: "Returns products from a Shopify store"
---

### Header

<ParamField header="x-api-key" type="string" required={true}>
    Parameter defines the Retailed private key to use.
</ParamField>

### Query Parameters

<ParamField query="index" type="string" optional={true} placeholder="mystore">
    Shopify slug of the store. Contact us if you need to get the slug of your store.
</ParamField>

<ParamField query="query" type="string" optional={true} placeholder="DV6773-220">
    Sneaker slug, name or sku
</ParamField>

### Response

<ResponseField name="data" type="array">
    Products data
</ResponseField>

```bash Example Request
curl --location --request GET 'https://app.retailed.io/api/v1/shopify/products?index=mystore&query=DM1275-250' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <token>' \
```
