---
title: "Releases"
api: "GET https://app.retailed.io/api/v1/sneakers/releases"
description: "Get all sneakers releases for a given date"
---

### Header

<ParamField header="x-api-key" type="string" required={true}>
    Parameter defines the Retailed private key to use.
</ParamField>

### Query Parameters

<ParamField query="released_at" type="string" optional={true} placeholder="2023-04-19">
    Sneaker released at.
</ParamField>

### Response

<ResponseField name="count" type="number">
    Indicates the number of sneakers returned
</ResponseField>

<ResponseField name="data" type="object">
    Array of sneakers
</ResponseField>

<ResponseField name="error" type="object">
    Not empty if an error occured
</ResponseField>

<ResponseField name="usage" type="object">
    API usage
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://app.retailed.io/api/v1/sneakers/releases?released_at=2023-04-19' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <token>' \
```

</RequestExample>
