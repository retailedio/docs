---
title: "360 Images"
api: "GET https://app.retailed.io/api/v1/sneakers/product_360"
description: "Get 360 images of sneakers"
---

### Header

<ParamField header="x-api-key" type="string" required={true}>
    Parameter defines the Retailed private key to use.
</ParamField>

### Query Parameters

<ParamField query="id" type="string" optional={true} placeholder="b5242fa5-48a4-444f-bd08-6a5db282b97b">
    Retailed internal ID
</ParamField>

<ParamField query="sku" type="string" optional={true} placeholder="DV6773-220">
    Sneaker code SKU
</ParamField>

### Response

<ResponseField name="data" type="list">
    Urls of 360 images
</ResponseField>

<ResponseField name="error" type="object">
    Not empty if an error occured
</ResponseField>

<ResponseField name="usage" type="object">
    API usage
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://app.retailed.io/api/v1/sneakers/product_360?sku=DH6927-140' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <token>' \
```

</RequestExample>
